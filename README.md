## Getting Started

### Dependencies

Will download all requirements to your virtual environment.

```
pip install -r requirements.txt
```

### Executing program

To initialize the database, run in your terminal:
```
flask --app flaskr init-db 
```

To start the local server; run in your terminal:
```
flask --app flaskr run
```

Visit http://127.0.0.1:5000/ in your browser

### Output

This web-app allows users to create an account, add, edit, & remove movies to thier collection
